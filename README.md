I've always wondered how much of the posts that I see on X are from paying vs. non paying users, since the majority always seem to be paying (that blue checkmark). 
Verifiable is a chrome extension that counts the number of posts scrolled, and how many of them was from no-badge/badged users.

I know, the UI sucks. You can thank 5.4 for that, this was made in <30s.
